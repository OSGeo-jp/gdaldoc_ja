GDALとは?

GDAL is a translator library for raster and vector geospatial data formats that is released under an MIT style Open Source ライセンス by the Open Source Geospatial Foundation. As a library, it presents a single raster abstract data model and single vector abstract data model to the calling application for all supported formats. It also comes with a variety of useful command line utilities for data translation and processing. The NEWS page describes the August 2024 GDAL/OGR 3.9.2 release.
